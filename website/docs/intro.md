# ยินดีต้อนรับสู่ n8n Mastery Course

คอร์สเรียน n8n แบบครบวงจรสำหรับผู้ที่ต้องการเชี่ยวชาญการใช้งาน n8n Workflow Automation และ Integration Platform ตั้งแต่ระดับพื้นฐานจนถึงขั้นสูง

## เกี่ยวกับคอร์สนี้

n8n Mastery Course เป็นส่วนหนึ่งของ Knowledge Fabric: Academy-as-Code System ที่ออกแบบมาเพื่อให้คุณเรียนรู้ n8n อย่างเป็นระบบและมีประสิทธิภาพ

### สิ่งที่คุณจะได้เรียนรู้

- **n8n Fundamentals**: การติดตั้ง n8n, การสร้าง Workflow แรก, การใช้ Nodes พื้นฐาน, และการเชื่อมต่อกับ Services ต่างๆ
- **Workflow Design**: การออกแบบ Workflow ที่มีประสิทธิภาพ, Data Flow, Error Handling, และ Best Practices
- **Automation Patterns**: การทำ Automation ต่างๆ เช่น Scheduled Tasks, Webhooks, API Integration, และ Event-Driven Workflows
- **Data Transformation**: การใช้ Expressions, Functions, และ Data Manipulation Nodes
- **Advanced Topics**: Custom Nodes, Webhooks, API Authentication, Production Deployment, และเทคนิคขั้นสูงอื่นๆ

## เริ่มต้นใช้งาน

### ความต้องการเบื้องต้น

- [Node.js](https://nodejs.org/en/download/) เวอร์ชัน 20.0 หรือสูงกว่า:
  - เมื่อติดตั้ง Node.js แนะนำให้เลือก checkbox ทั้งหมดที่เกี่ยวข้องกับ dependencies

### เริ่มต้นพัฒนา

1. ไปที่โฟลเดอร์ website:
```bash
cd website
```

2. ติดตั้ง dependencies:
```bash
npm install
```

3. เริ่ม development server:
```bash
npm run start
```

เว็บไซต์จะเปิดที่ http://localhost:3000 และจะ reload อัตโนมัติเมื่อคุณแก้ไขไฟล์

## โครงสร้างคอร์ส
